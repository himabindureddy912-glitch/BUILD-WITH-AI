# BUILD-WITH-AI
Innvotex hackathon
# PROBLEM STATEMENT:
Managing seating arrangements manually for exams, events, or classrooms is time-consuming and error-prone. 
- Poor space utilization
- Difficulty in maintaining rules (like distancing or grouping)
- Last-minute confusion and inefficiency
# PROPOSED SOLUTION:
We propose an AI-based seating arrangement system that automatically generates optimal seating plans based on constraints such as capacity, spacing rules, preferences, and special requirements.
# KEY FEATURES:
- Automated seat allocation
- Constraint-based arrangement (distance, grouping, priority)
- Dynamic updates (handles last-minute changes)
- Visualization of seating layout
- Export seating plan (PDF/Excel)
- Sends Email/SMS to students with:
   Room number
   Seat number
   Exam details
#  TECH STACK:
- Frontend: HTML, CSS, JavaScript 
- Backend: Python 
- AI Logic: Python (NumPy, Pandas, Scikit-learn)
- Database: Microsoft Excel
  # HOW IT WORKS:
1. Admin inputs:
   - Number of students/participants
   - Seating layout (rows, columns)
   - Constraints (distance, groups, etc.)
2. System processes data using AI algorithm
3. Generates optimized seating plan
4. Displays and allows download
# USE CASES:
- Exam hall seating
- Classroom management
- Conferences & events
- Office space allocation
# ADVANTAGES:
- Saves time and effort
- Reduces human error
- Ensures fairness and compliance
- Scalable for large datasets
